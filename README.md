# Moon Dataset - SVM Classification

Binary classification on the non-linear moon-shaped dataset using Support Vector Machine (SVM).

## What this project does
- Generates moon-shaped data using sklearn's `make_moons`
- Visualizes the dataset using matplotlib scatter plot
- Splits data into training and testing sets (70/30)
- Trains an SVM model with linear kernel
- Evaluates model accuracy on test data

## Libraries Used
- `scikit-learn` — dataset generation, SVM model, train-test split, accuracy score
- `pandas` — data handling
- `matplotlib` — data visualization

## Dataset
Generated using `sklearn.datasets.make_moons` with 200 samples.

## Model
- Algorithm: Support Vector Machine (SVM)
- Kernel: Linear
- Test Size: 30%
